# Проект "Игры разума"
Включает в себя 5 игр:
- чет / нечет brain-even
- подсчет brain-calc
- простое / непростое число brain-prime
- найти НОД brain-gcd
- найти пропущенное число brain-progression

## Установка
```shell
git clone git@github.com:mileevamaria/python-project-49.git
brew install uv
make build
make package-install
```
